# Chess-Engine
Chess Engine is a work in progress project that involved experimentation with chess programming.

the project remains unfinished due to loss in interest.

What was implemented was:
  board representation,
  psuedolegal move generation,
  and several organizing struct objects.

Issues: it's heavily incomplete, missing:
  king safety functions,
  full legal move generation,
  minimax with alpha-beta pruning,
  and certain optimisations.
